git- 更新代码
```
//更新指定仓库代码
repo sync system/core
//开启master分支
repo start master system/core
```
- 提交代码
```
//查看状态
git status
//查看远程分支
git branch -a
//add
git add xxx
//commit
git commit -m
//push
git push origin master:refs/for/master
//修改后再次提交（避免产生新的提交记录)
git commit --amend
git push origin master:refs/for/master
//遇到提示，按ctlr+shift+x退出
