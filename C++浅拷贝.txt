防止浅拷贝，当类中有指针成员的时候，就要写上私有的拷贝构造函数和等号运算符的重载

private 

   A(const A&)

   A& operater =（const A&）

只写定义，不写实现，因为就是不想用这个函数。

赋值过程中默认的只是简单的赋值，会有两个指针指向同一空间的情况，一个释放，另一个就无法释放了。


需要注意的点：
1. 深拷贝和浅拷贝发生在类成员中包含有指针域的时候。

2. =操作符和对象的初始化是两种不同的东西！！


// 定义obj2并且用obj1初始化
Name obj1("obj1");
Name obj2 = obj1; // 进行的是拷贝构造函数

// 把obj3复制给obj2，用等号操作
Name obj3("obj3.....");
obj2 = obj3;      // 进行的是等号操作
